# Portfólio Perícia Digital

Portfólio profissional demonstrando expertise em perícia digital, análise de documentos eletrônicos e verificação de assinaturas digitais.

## Visão Geral

Este portfólio demonstra competências técnicas em:

- **Análise de Assinaturas Digitais ICP-Brasil**
- **Verificação de Autenticidade de Documentos**
- **Validação de Certificados Digitais**
- **Detecção de Adulterações em Documentos**
- **Análise Forense de Metadados**

## Sobre Este Portfólio

Este repositório contém estudos de caso, metodologias e modelos profissionais que demonstram expertise em perícia digital e análise de documentos eletrônicos. Todos os casos apresentados são anonimizados e utilizam apenas informações públicas ou autorizadas.
